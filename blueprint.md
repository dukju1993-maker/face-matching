
# Blueprint: AI Gwansang & Saju Dating Service

## Overview

This project is a web-based dating service that matches users based on the Korean traditions of *Gwansang* (face reading) and *Saju* (Four Pillars of Destiny). Users will provide their personal information, including a facial photo and birth details, as well as a description of their ideal partner. The service will then use AI-powered analysis to find a compatible match.

## Initial Plan

1.  **Create the basic HTML structure (`index.html`)** for the user interface, including input fields for personal data and ideal type.
2.  **Style the page (`style.css`)** to be visually appealing and user-friendly, following modern design principles.
3.  **Add interactivity (`main.js`)** to handle form submissions and user interactions.
4.  **Implement a matching logic** (to be developed) that will take user data and find the best match.
